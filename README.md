Using on Windows
================
```php
1.open cmd
2.cd /path/of/webserver   such as  c:\xampp\htdocs
3.git clone https://github.com/prawee/yiiapp.git your-project-name
4.cd your-project-name folder
5.composer self-update
6.composer --prefer-dist update
```
